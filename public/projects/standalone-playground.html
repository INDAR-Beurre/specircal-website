Puter.js Architecture

Puter.js uses the "User-Pays" model. Instead of developers bearing the costs of AI API calls, users handle their requests via Puter's client-side runtime environment.

This sandbox uses:

  • puter.ai.chat() for text processing, code generation, reasoning, and image input analysis.
  • puter.ai.txt2img() for visual generation using Qwen Image 2.0.
  • No backend API Keys, making it highly secure and zero-maintenance.
