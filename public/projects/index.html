Offline by design.
Online when possible.

Traditional web architectures fail the moment connection degrades. Aura DB shifts the source of truth to the client device. Reads take 0ms. Writes take 0ms. Transactions succeed instantly offline and queue for server merge.

  • Replicated Cache: Automatic client database syncing.
  • Optimistic Queries: Immediate UI feedback without server roundtrips.
  • Intelligent Reconnect: Incremental delta synchronizations.

Mathematical conflict resolution.

Say goodbye to version mismatch errors and destructive overwrites. Aura DB implements conflict-free replicated data types (CRDTs) under the hood. Concurrency issues resolve themselves programmatically on convergence, keeping data coherent across every active node.

  • LWW-Register: Last-Write-Wins timestamps resolution.
  • Grow-Only Counter: Additive counters safe from duplicate merges.
  • Custom Merge Hooks: Intercept and resolve application logic.

Developer Friendly

Works with simple native methods.

Zero complicated setup steps. Initialize in three lines, query reactively.

import { AuraDB } from '@auradb/client';

// Initialize the DB and automatically load the disk store
const db = await AuraDB.open({
  name: 'user_workspace',
  syncUrl: 'wss://sync.auradb.io/v2'
});

// Sub-millisecond write (instant local update, async sync)
const recordId = await db.collection('tasks').insert({
  title: 'Design landing hero layout',
  completed: false,
  updatedAt: Date.now()
});

// Reactive queries notify your UI automatically
db.collection('tasks')
  .query({ completed: false })
  .subscribe(tasks => {
    updateUI(tasks);
  });
import { useQuery, useMutation } from '@auradb/react';

export default function App() {
  // Hook auto-subscribes to local DB state transitions
  const { data: tasks, loading } = useQuery('tasks', { completed: false });
  const [insertTask] = useMutation('tasks');

  return (
    <div className="app-container">
      <button onClick={() => insertTask({ title: "Quick task" })}>
        Add Task
      </button>
      <ul>
        {tasks.map(t => <li key={t._id}>{t.title}</li>)}
      </ul>
    </div>
  );
}
{
  "collection": "tasks",
  "encryption": true,
  "properties": {
    "title": { "type": "string" },
    "completed": { "type": "boolean" },
    "updatedAt": { "type": "number" }
  },
  "indexes": ["completed", "updatedAt"]
}
